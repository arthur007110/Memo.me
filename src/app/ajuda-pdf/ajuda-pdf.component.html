Tutorial

Antes de começar a criar os modelos de PDF se faz necessário levar em consideração algumas observações:

  • Deve-se começar cada linha com: &&.
  • Caso o campo "Tamanho da Fonte" não seja preenchido será atribuído um valor padrão (tamanho 12).
  • Para salvar o modelo é necessário escolher um nome válido, um que ainda não esteja em uso.
  • Para adicionar/remover uma imagem basta clicar no botão "Adicionar/Remover Imagem" e preencher os campos referentes a suas propriedades (largura, altura e posição no arquivo com relação aos eixos x e y). Caso algum desses campos não seja preenchido será atribuído um valor padrão ou o valor original, no que se refere a suas dimensões.
  • Para adicionar uma imagem copie sua url codificada na base 64.
  • Para escolher a posição dos campos referentes aos atributos de memorando (mensagem, assunto, data de emissão, número do memorando, nome e siape do usuário emissor e os nomes dos setores emissor e de destino) deve-se usar um dos comandos da lista abaixo.

Durante o processo de criação para visualizar a situação do documento basta clicar em "Visualizar". Por fim, após concluído o documento basta clicar em "Salvar".

Comandos

  • &MENSAGEM: local onde ficará o corpo do memorando
  • &ASSUNTO: local onde ficará o assunto do memorando
  • &DATA: local onde ficará a data de envio do memorando
  • &NUMERO: local onde ficará o número do memorando
  • &NOME: local onde ficará o nome do usuário
  • &SIAPE: local onde ficará a siape do usuário
  • &SETOREMISSOR: local onde ficará o nome do setor emissor do memorando
  • &SETORDESTINO: local onde ficará o nome do setor destinatário do memorando
